How to use TextTally

  1. Select any text on a webpage
  2. Right-click on the selected text
  3. Click "Analyze with TextTally"
  4. View your text statistics!

Features

  • 📊 Word & character counts
  • ⏱️ Reading time estimate
  • 📚 Readability score
  • ☁️ Keyword analysis
  • ⚠️ Long sentence alerts
  • 💾 Export options
